# Flood Prediction System using Machine Learning

## Overview

This project predicts the possibility of floods using Machine Learning algorithms. It analyzes historical weather data such as rainfall, cloud visibility, and seasonal rainfall patterns to predict flood occurrence. The project helps authorities and disaster management teams take early preventive actions.

## Features

- Flood Prediction using Machine Learning
- Weather Data Analysis
- Early Flood Warning
- User-Friendly Web Application
- High Prediction Accuracy

## Technologies Used

- Python
- Flask
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
- XGBoost

## Machine Learning Algorithms

- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- XGBoost

## Accuracy

The XGBoost model achieved an accuracy of **96.55%**.
